## Project Title: AD 312 Integrating React Hook Form with TanStack Query and Mock REST APIs
#### By: Amde Wubshet

### Breif Description
 integrate React Hook Form with TanStack Query (React Query) to handle data fetching, caching, and mutations against a local Mock REST API (JSON Server). 

### Commands 
After installing the repo, navigate into your project directory: *cd profile-assignment*. Then split the terminals: 
- On one terminal, start the json server: *npx json-server db.json --port 3001*. If you don't do this then the local app launch will stay loading infinetly  
- On the other server, start the development server: *npm run dev*.
